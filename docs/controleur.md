# Guide Contrôleur

Le profil **Contrôleur terrain** permet de préparer et saisir les constats sur le terrain, avec rattachement aux dispositifs et pièces jointes photo.

![Capture annotée — espace contrôleur TLPE](assets/controleur-workspace.svg)

## Constats terrain et pièces jointes

Depuis la page **Contrôles terrain** :
- créer un constat ;
- rattacher le constat à un dispositif existant ou préparer la création d’une fiche ;
- joindre les photos terrain ;
- suivre les états de synchronisation selon le contexte de connexion.

## Carte et repérage

Le menu **Carte des dispositifs** complète l’activité terrain :
- localisation d’un dispositif ;
- contrôle visuel de zone ;
- préparation des vérifications sur site.

## Suites métier

Une fois le constat clôturé, les équipes habilitées peuvent :
- générer un rapport de contrôle ;
- proposer une rectification ;
- lancer un redressement ou un contentieux.

## Application mobile native (phase ultérieure)

L'application mobile iOS / Android demandée par l'US7.2 n'est **pas** incluse dans le MVP web actuel.
La trajectoire prévue, les dépendances externes (comptes stores, stratégie MDM, notifications push) et le contrat d'intégration avec le backend sont détaillés dans la [feuille de route mobile native](mobile-native-roadmap.md).
